# Scoped 🎯
Turn intention into completed learning.

An AI-powered intent-locking learning companion built for the 
DDS AI Application Building Challenge 2026.

What is Scoped?
You declare what you want to learn and for how long.
Scoped builds you a real-time AI briefing, locks you into 
a focused session, and ends with a Knowledge Pulse quiz 
that proves you actually retained something.

Tech Stack
- Python 3.11
- OpenAI GPT-4o API
- Tavily Search API
- Streamlit
